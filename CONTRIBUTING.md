# Contributing to Dhook

Thank you for your interest in contributing to **dhook**! We welcome contributions from the community and appreciate your effort to help improve this project.

## How to Contribute

### 1. Fork and Clone the Repository

1. Fork this repository by clicking the **Fork** button on the top right of the GitHub page.
2. Clone your forked repository to your local machine:

   ```bash
   git clone https://github.com/YOUR_USERNAME/dhook.git
   cd dhook
   ```

3. Add the original repository as an upstream remote:

   ```bash
   git remote add upstream https://github.com/alisheraxmedov/dhook.git
   ```

### 2. Create a Feature Branch

Before making any changes, create a new branch from the `main` branch:

```bash
git checkout main
git pull upstream main
git checkout -b feature/your-feature-name
```

### 3. Make Your Changes

Implement your feature, bug fix, or improvement. Please ensure your code follows the existing code style and conventions.

### 4. Commit and Push

Commit your changes with a clear and descriptive commit message:

```bash
git add .
git commit -m "Add: brief description of your changes"
git push origin feature/your-feature-name
```

### 5. Submit a Pull Request

1. Go to your forked repository on GitHub.
2. Click **"Compare & pull request"**.
3. Ensure the base branch is set to `master` of the original repository.
4. Provide a clear title and description for your pull request.
5. Submit the pull request for review.

## Reporting Issues and Suggestions

If you encounter any bugs, have questions, or would like to suggest improvements, please open an issue in the [Issues](https://github.com/alisheraxmedov/dhook/issues) section.

When creating an issue, please include:
- A clear and descriptive title
- Steps to reproduce the problem (if applicable)
- Expected vs. actual behavior
- Your environment details (OS, Dart version, etc.)

## Code of Conduct

Please be respectful and considerate in all interactions. We are committed to maintaining a welcoming and inclusive community.

Thank you for contributing to **dhook**!
